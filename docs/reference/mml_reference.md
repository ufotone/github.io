# MML 記法リファレンス

| 記号 | 意味 | 例 |
|------|------|------|
| C D E F G A B | 音符 | C4 |
| + / # / - | シャープ / フラット | C+ |
| R | 休符 | R8 |
| 数字 | 音長 | C8 |
| . | 付点 | C4. |
| O | オクターブ | O4 |
| < / > | オクターブ上下 | >C |
| L | デフォルト音長 | L8 |
| T | テンポ | T140 |
| V | 音量 | V12 |
| @ | 音色 | @3 |
| & | タイ | C4&C4 |
