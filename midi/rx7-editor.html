⚠ Before SysEx will work, prepare the RX7:
  1. Device#: MIDI → JOB 07 → +1/YES → set to 01 (default is off and blocks all SysEx). Set the IO section RX7 Device# Value to the same number.
  2. Memory Protect OFF: UTILITY → JOB 03 → -1/NO (required for any SysEx write — Note# Assign, bulk send, etc.). Use the IO section button as a shortcut.
  3. Channel Message ON: MIDI → JOB 01 → +1/YES (needed to send/receive Note On/Off, Program Change, etc.).
